Question-1: What are the different ways to select an element in the DOM?

There are several ways like getElementById, getElementsByClassName, getElementsByTagName to select an element in the DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML processes HTML tags, innerText gets only visible text, textContent gets all text including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is type of thing when event is added to the main element to manage other events corresponding to it.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is the process where event starts form target element.

Question-5: How do you create, add, and remove elements using JavaScript?

For creating, adding and removing elements using JavaScript we need to use document.createElement, appendChild and removeChild tags.
